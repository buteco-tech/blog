  • NVIDIA anuncia driver 358 (beta) para Linux, Solaris e FreeBSD

Construído sobre o driver 355, a série 358 acrescenta mais peças ao quebra-cabeças da interface com DRM/KMS (Direct Rendering Manager/Kernel Mode Setting) e continua melhorando o suporte ao Mir/Wayland.
