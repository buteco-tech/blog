    Com a maior parte dos projetos sendo hospedados no GitHub, incluindo vários da própria Google, a Google resolveu tirar do ar o seu serviço de hospedagem de código fonte, conhecido como Google Code. Para quem não o conhece, o Google Code é um serviço de hospedagem de código muito similar ao GitHub ou Bitbucket.
