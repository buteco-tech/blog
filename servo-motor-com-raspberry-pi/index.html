Servo motor com Raspberry Pi

Fala galera! Hoje vamos ver um exemplo de como trabalhar com um servomotor no Raspberry PI usando a linguagem Python.

Antes de mais nada gostaria de agradecer ao nosso patrocinador Baú da Eletrônica por ter fornecido o servomotor usado nesse tutorial. Primeiramente precisaremos conectar nosso servo motor as portas de GPIO, será necessário 3 portas, uma para comando (GPIO21), uma para energia (5v) e uma para terra (GND), no meu caso eu utilizei os pinos 40, 02 e 39 respectivamente, o modelo que utilizei foi o Raspberry Pi B+, abaixo pode-se ver a imagem das conexões.

Conforme a imagem acima, no caso do servomotor Tower Pro SG90, o fio violeta é o 5V e está conectado no pino 2, o fio azul que é o de comando, está no pino 40, e por fim o fio dourado que é o terra está no pino 39. Uma vez conectado podemos acessar nosso Raspberrypi e salvar o código abaixo em um arquivo qualquer, comentários em cada linha explicam o que cada comando faz.

Este é um post breve para mostrar como trabalhar com servo motores, testem o código e vejam se funciona com vocês, comentem e avaliem e até a próxima pessoal.
