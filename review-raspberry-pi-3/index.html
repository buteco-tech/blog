Review Raspberry Pi 3

Alexandre Vicenzi em Embarcados e Reviews
10/05/2016 · 2 minutos de leitura

Atenção

Este artigo foi migrado do Wordpress e pode conter erros de formatação.

Caso encontre algum problema deixe um comentário que iremos resolver o mais breve possível.

O Raspberry Pi 3 acabou de chegar na FILIPEFLOP e esta nova versão traz novidades em relação a versão anterior. As maiores novidades são o WiFi e Bluetooth integrado.

Na tabela abaixo podemos observar a diferença entre a versão 2 e a 3.

RPi 3 Model B RPi 2 Model B
Núcleos Quad Core Quad Core
CPU Cortex A53 1.2 GHz 64 bits Cortex A7 900 MHz
Arquitetura ARM (v8) ARM (v7)
RAM 1 GB SDRAM 1 GB SDRAM
GPU 400 MHz VideoCore IV 250 MHz VideoCore IV
Armazenamento micro-SD micro-SD
Ethernet 10/100 10/100
WiFi 802.11n -
Bluetooth 4.1 BLE -
Assim como o Raspberry Pi 2, o 3 possui 4 portas USB, 40 pinos GPIO, suporte a HDMI, interface CSI para a câmera e DSI para o display. O Raspberry Pi 3 é compatível com as versões 1 e 2, sendo a única diferença visível a posição dos LEDs, que mudaram de lado para dar lugar a antena WiFi, ficando próximo ao conector de alimentação micro USB.

Considerações Finais

A nova CPU e GPU com certeza trazem um ganho de desempenho para esta nova versão do RPi, especialmente para aqueles que pretendem montar um console de games retrô ou um mini PC.

A presença de Bluetooth e WiFi integrados evita a aquisição de itens adicionais para tarefas simples.
